CSS Interview Questions

Q1). Diffrentiate between flex-row and column ?

In the display: flex property, there are two values for flex direction. In flex-row, the items start from the edge of the page it means it works on the main axis from left to right whereas in flex-column the items are aligned from top to bottom.

Q2). Explain inline, internal and external stylesheet?

The order of priority in Cascading Style Sheet is as follows:

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Inline CSS: Inline css is the type of css in which style tag is placed inside the html tags.

Internal CSS: Internal CSS is the type of css in which style tag is placed inside the head tag.

External CSS: External css is the type of css in which we create an external css file and link it with the html page.

Q3). Justify-content allows you to do what?

Jutsify-content defines how the browser distributes the space between and around content items along the main-axis which is also called as primary or major axis of a flex container and the minor axis or secondary axis of the container.

Q4). Differentiate between absolute and relative positioning?

In CSS,the position property specifies the type of positioning method used for an element(static, relative, fixed, absolute or sticky)

An element with position:relative ; is positioned relative to its normal position whereas Absolute positioning is positioned relative to the nearest positioned ancestor.

Q5). What is grid template column used for?

In CSS, under display as grid, Grid template column specifies the number of columns and the size of columns of the grid.

Q6).What is z-index in css?

In CSS,z-index is the property that specifies the stack order of an element. An element can have a positive or negative stack order. Z-index works only when positions are taken in use.

Q7). What is the difference between padding and margin?

In CSS, margin is the space around an element's border, while padding is the space between an element's border and the element's content. This can be seen in the box-model.
